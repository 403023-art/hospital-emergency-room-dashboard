# Hospital Emergency Room Dashboard

## Project Overview
This is an interactive Excel dashboard designed to track and analyze key performance metrics for a Hospital Emergency Room. It provides monthly insights into patient volumes, wait times, satisfaction levels, and demographic trends.

## Tech Stack & Advanced Features Used
* **Power Query:** Used for data extraction, cleaning, and transformation (ETL).
* **Power Pivot:** Utilized to build a relational data model and handle data relationships.
* **DAX Formulas:** Developed to calculate complex KPIs like Average Wait Time and Patient Satisfaction Scores.
* **Dynamic UI:** Features custom slicers and conditional formatting for an interactive user experience.

## How to Use the Project
1. Download the `excelfullfledged_project.xlsx` file from this repository.
2. Open it in Microsoft Excel (ensure Power Pivot and Power Query add-ins are enabled).
3. Use the monthly slicers on the left side of the dashboard to filter and interact with the data dynamically.
